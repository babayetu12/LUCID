# LUCID - Dream Tracker

A fully responsive, dark-mode native web application for logging, tagging, and analyzing lucid dreams, sleep patterns, and REM cycles.

## Features
- **Comprehensive Logging:** Track vividness, length, and overall rating.
- **Smart Tags:** Categorize people, places, objects, emotions, actions, and themes.
- **Sleep Metrics:** Input Garmin/Bevel sleep scores and total/REM sleep times.
- **Advanced Analytics:** Dynamic correlation between vividness and lucidity, highest-rated dream elements, and best sleeping days.
- **Local First:** Currently runs entirely on LocalStorage.

## Tech Stack
- HTML5
- Tailwind CSS (via CDN)
- Vanilla JavaScript
- FontAwesome (Icons)

## Upcoming (Supabase Integration)
This project is currently being prepared for persistent cloud storage via Supabase, allowing cross-device syncing of your dreams and analytics.
